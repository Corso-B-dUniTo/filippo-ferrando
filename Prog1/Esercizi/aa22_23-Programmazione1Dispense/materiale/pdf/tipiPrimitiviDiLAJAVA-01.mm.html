TIPI DI DATI PRIMITIVI

(da The Java Tutorials)

  • Numeri interi

    • byte: The byte data type is an 8-bit signed

      two's complement integer. It has a minimum

      value of -128 and a maximum value of 127

      (inclusive).

      Dichiarazione. 

      byte b = 100; // entro il range

      short: The short data type is a 16-bit signed

      two's complement integer. It has a minimum

      value of -32,768 and a maximum value of 32,767

      (inclusive).

      Dichiarazione. 

      short s = 10000; // entro il range

      int: The int data type is a 32-bit signed

      two's complement integer. It has a minimum

      value of -2,147,483,648 and a maximum value

      of 2,147,483,647 (inclusive). For integral values,

      this data type is generally the default choice

      unless there is a reason (like the above) to choose

      something else.

      Dichiarazione. 

      int i = 100000; // entro il range

      long: The long data type is a 64-bit signed

      two's complement integer. It has a minimum

      value of -9,223,372,036,854,775,808 and a

      maximum value of 9,223,372,036,854,775,807

      (inclusive).

      Dichiarazione. 

      long i = 100000L; // letterale int seguito da L
long j = 200000l; // letterale int seguito da l

    Numeri in

    virgola mobile

    • float: The float data type is a single-precision

      32-bit IEEE 754 floating point.

      Definizione. 

      float f1  = 123.4F
float f1  = 123.4f;

      double: The double data type is a

      double-precision 64-bit IEEE 754

      floating point:

      • 53 bit per la rappresentazione della mantissa,
      • 11 bit per la rappresentazione dell'esponente.

      Definizione. 

      double d1 = 123.4;
// same value as d1, but in scientific notation
double d2 = 1.234e2;
double d3 = 1.234E-2; // 0.01234
double d4 = -1.234E-2; // -0.01234
double d5 = 123.4D;double d6 = 123.4d;

    Booleani

    • boolean: The boolean data type has

      only two possible values: true and false.

      Use this data type for simple flags that

      track true/false conditions. 

      Dichiarazione. 

      boolean result = true;

    Caratteri

    • char: The char data type is a single 16-bit

      Unicode character. It has a minimum value

      of '\u0000' (or 0) and a maximum value of

      '\uffff' (or 65,535 inclusive).

      Dichiarazione. 

      char capitalC = 'C';
